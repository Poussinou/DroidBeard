About

What is DroidBeard?

DroidBeard is a free, open-source, lightweight and easy to use Android app for managing your SickBeard installation either at home or remotely.

Why is the app free?

The developers of SickBeard were kind enough to release their product for free, so it only seems fair that a spin off project should follow suit and be for the good of the community rather than monetary gain.

How can I contribute?

If you're interested in contributing you can do so by reporting bugs you find via The GitHub Issue List, contributing code on GitHub or by simply spreading the word to your friends!
